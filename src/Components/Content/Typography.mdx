# Typography

## Headings

<Story id="typography--headings" />

```html
<h1>H1 title</h1>
<h2>H2 title</h2>
<h3>H3 title</h3>
<h4>H4 title</h4>
<h5>H5 title</h5>
<h6>H6 title</h6>
```

You can use classnames to emulate headings on any other element:

  <Story id="typography--class-names" />

```html
<div class="h1">H1 style</div>
<div class="h2">H2 style</div>
<div class="h3">H3 style</div>
<div class="h4">H4 style</div>
<div class="h5">H5 style</div>
<div class="h6">H6 style</div>
```
